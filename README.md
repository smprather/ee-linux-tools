# ee-linux-tools
Linux tools compiled to common EDA linux distro targets (EL7/8+). Configured to work offline when applicable.

# Architectures
* x86_64
* ARM
 
# Platforms
* EL7, 8, 9
  * Base images: Centos7, Alma for the rest
* Maybe Suse if requested
